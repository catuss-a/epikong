epikong
=======

epikong is a little game coded in 2days. it is developed in C using the SDL library.

FPS are not handled so it depends on your CPU/GC performances :/

gameplay
=======

- LEFT and RIGHT keys to move
- SPACE key to jump

FIND THE CARROT AND GET OUT /////////////////////////////////

Have fun!
